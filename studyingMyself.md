## ngModule의 생성

`ng generate module todo`

`ng g c todo/todos --module todo/todo.module.ts --export`

c - component 약어
todo 라는 폴더 아래 todos 
--module 특정 모듈의 지정
todo/to.module.ts 를 특정 모듈로써 지정
--export 다른 데서 사용할 수 있다는 의미의 export 추가
